HTML Basics & Fortgeschritten: Ein Tutorial für Anfänger

Willkommen zu unserem HTML-Tutorial! In diesem Tutorial werden wir die Grundlagen und fortgeschrittene Techniken von HTML behandeln. Multimedia-Inhalte sind ein wesentlicher Bestandteil moderner Webseiten, und wir werden zeigen, wie man sie effektiv einbindet.

Zeilenumbrüche

Ein Zeilenumbruch kann mit dem <br>-Tag eingefügt werden.

<p>Dies ist ein Satz.<br>Dies ist der nächste Satz.</p>

Absätze

Absätze werden mit dem <p>-Tag erstellt.

<p>Dies ist ein Absatz.</p>

Horizontale Linien

Eine horizontale Linie kann mit dem <hr>-Tag eingefügt werden.

<hr>

Überschriften

HTML bietet sechs Ebenen von Überschriften, von <h1> bis <h6>.

<h1>Hauptüberschrift</h1>
<h2>Unterüberschrift</h2>
<h3>Unterunterüberschrift</h3>

Kommentare

Kommentare werden zwischen <!-- und --> gesetzt.

<!-- Dies ist ein Kommentar -->

Textformatierungen

Text kann fett, kursiv oder unterstrichen dargestellt werden.

<b>Fetter Text</b>
<i>Kursiver Text</i>
<u>Unterstrichener Text</u>

Spezielle Textformate

Code kann mit dem <code>-Tag und vorformatierter Text mit dem <pre>-Tag dargestellt werden.

<code>console.log('Hallo Welt');</code>
<pre>Vorformatierter Text</pre>

Universelle Tags

Das <div>-Tag wird verwendet, um Block-Elemente zu gruppieren, und das <span>-Tag wird verwendet, um Inline-Elemente zu gruppieren.

<div>Block-Inhalt</div>
<span>Inline-Inhalt</span>

Bilder einbinden

Bilder können mit dem <img>-Tag eingebunden werden. Beispiel:

<img src="https://example.com/bild.jpg" alt="Beispielbild" width="300" height="200">

Videos einbinden

Videos können mit dem <video>-Tag eingebunden werden. Beispiel:

<video width="320" height="240" controls>
  <source src="film.mp4" type="video/mp4">
  Ihr Browser unterstützt das Video-Tag nicht.
</video>

Musik einbinden

Musik kann mit dem <audio>-Tag eingebunden werden. Beispiel:

<audio controls>
  <source src="musik.mp3" type="audio/mpeg">
  Ihr Browser unterstützt das Audio-Tag nicht.
</audio>

IFrame einbinden

Ein IFrame kann verwendet werden, um eine andere Webseite innerhalb der aktuellen Seite anzuzeigen. Beispiel:

<iframe src="https://example.com" width="600" height="400"></iframe>

Zusammenfassung

In diesem Tutorial haben wir die grundlegenden und fortgeschrittenen HTML-Elemente behandelt, die für das Design moderner Webseiten wichtig sind. Durch das Verständnis und die Anwendung dieser Elemente können Sie ansprechende und strukturierte Webseiten erstellen.
